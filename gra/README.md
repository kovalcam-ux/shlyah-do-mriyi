# 🥊 Загін Носик — гра-файтинг (прототип, Етап 1)

Браузерна 2D-гра один-на-один (стиль Mortal Kombat) для сайту притулку.
Рушій — **Phaser 4** (підключений із CDN). Працює на ПК і телефоні в браузері.

## ▶️ Як відкрити

- **Онлайн (після пушу):** `https://kovalcam-ux.github.io/shlyah-do-mriyi/gra/`
- **Локально:** запусти простий сервер у корені репозиторію і відкрий `/gra/`:
  ```bash
  npx serve .
  ```
  (просто відкрити файл через `file://` не можна — браузер блокує завантаження скриптів).

> Потрібен інтернет: рушій Phaser підвантажується з CDN.

## 🎮 Керування

**ПК (клавіатура):**
- Рух: `← →` або `A` `D`
- Стрибок: `↑`  •  Присід: `↓`
- Удар рукою: `J`  •  Удар ногою: `K`  •  Блок: `L`

**Телефон (сенсор):**
- Ліворуч — віртуальний джойстик (рух, угору = стрибок, униз = присід).
- Праворуч — кнопки: **УДАР**, **НОГА**, **БЛОК**.
- Грати треба горизонтально. Якщо телефон вертикально — зʼявиться підказка «Поверни телефон».

## 🩸 Перемикач крові

У головному меню кнопка **«Кров: ВИМК/УВІМК»**.
- **Вимк (за замовчуванням, безпечний режим):** від ударів — золоті іскри й зірочки.
- **Увімк:** стилізовані червоні бризки.

Вибір зберігається у браузері. Кров і удари — **тільки між бійцями**. Жодного насильства над тваринами.

## 🏆 Правила бою

- Поєдинок до **2 виграних раундів** (best of 3).
- Раунд — **90 секунд**. Якщо час вийшов, перемагає той, у кого більше здоровʼя.
- Перемога гравця → **сцена порятунку**: клітку розбито, суперника забирає поліція,
  виходить намальована тваринка і зʼявляється заклик ділитися + кнопки соцмереж.

## 🧩 Структура файлів

```
gra/
├── index.html          сторінка гри (канвас, сенсорні кнопки, підказка орієнтації)
├── style.css           стилі сторінки і мобільного керування
├── js/
│   ├── config.js       ⭐ усі налаштування + ШЛЯХИ ДО СПРАЙТІВ (заглушки тут)
│   ├── ui.js           кнопки інтерфейсу
│   ├── effects.js      ефекти ударів (кров / зірочки)
│   ├── input.js        клавіатура + сенсорний джойстик
│   ├── fighter.js      боєць (логіка руху, ударів, блоку, здоровʼя)
│   ├── ai.js           простий ШІ суперника
│   ├── main.js         запуск Phaser (масштаб FIT, фізика)
│   └── scenes/
│       ├── boot.js     генерує заглушки-прямокутники бійців
│       ├── menu.js     меню + перемикач крові
│       ├── fight.js    бій, раунди, дві шкали HP, таймер
│       └── rescue.js   екран перемоги (сцена порятунку)

```

## 🎨 Як підставити справжній піксель-арт (наступний крок)

Зараз бійці — кольорові прямокутники-заглушки. Код уже готовий до спрайтів.
Достатньо у `js/config.js` у блоці `FIGHTERS`:

```js
player: {
  useSprites: true,                 // увімкнути спрайти
  sheet: 'assets/player.png',       // твій спрайтшит
  frameW: 96, frameH: 96,           // розмір кадру
  animations: {
    idle:  { frames: [0,1,2,3], rate: 6,  repeat: -1 },
    walk:  { frames: [4,5,6,7], rate: 10, repeat: -1 },
    punch: { frames: [8,9],     rate: 14, repeat: 0  },
    // ...
  },
}
```

Тваринку у сцені порятунку (`scenes/rescue.js`, метод `_drawDog`) теж можна замінити справжнім спрайтом.

## 📦 Як винести гру в окремий репозиторій

Гра самодостатня — це проста папка `gra/`. Щоб зробити окремий репозиторій:

1. Створи новий порожній репозиторій на GitHub (напр. `zahin-nosyk`).
2. Скопіюй уміст папки `gra/` у корінь нового репозиторію.
3. Увімкни **Settings → Pages → Deploy from branch → main** — гра відкриється за адресою репозиторію.

(У файлі `index.html` посилання «← На сайт притулку» веде на `../index.html` — у окремому репозиторії підправ його на адресу головного сайту.)

## ⚠️ Що НЕ входить у цей етап

Один боєць, один суперник (Базарний Ділок), без ростера, без боса, без аркадних сходів,
без таблиці лідерів, без суперприйому та шкали енергії, без Supabase. Це свідомо мінімальний прототип.
