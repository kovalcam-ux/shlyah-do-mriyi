# 🦊 Шлях до мрії — трекер цілей притулку для тварин

**Гейміфікований трекер цілей** благодійного притулку «Шлях до мрії», де живуть **~75 безпритульних собак і ~80 безпритульних котів**. Ми з дружиною доглядаємо їх самостійно — цей застосунок допомагає нам планувати щоденну роботу, рухатись до великої мрії та залучати людей до допомоги.

## 🐾 Про притулок

Щодня ми годуємо, лікуємо та доглядаємо понад 150 тварин. Наша велика мрія — побудувати повноцінний комплекс для порятунку тварин. Кожна гривня і кожна пара рук наближають нас до неї.

## ✨ Що вміє застосунок

- 🎯 **Цілі** — AI (Claude) розбиває велику ціль на конкретні кроки, кожен крок дає монети та досвід.
- 🔁 **Щодня** — щоденна рутина (годування, прибирання, ліки) зі скиданням щоранку, серією днів і відповідальними.
- 🌟 **Мрія** — великий грошовий збір із прогрес-баром і внесками.
- 🎁 **Нагороди** — система самозаохочення за зароблені монети.
- 💪 **Навички** — облік сильних сторін, які AI враховує у порадах.
- 📣 **Поділитися прогресом** — генератор картинок для Instagram/Facebook (пост 1080×1080 та сторіс) із закликом до донатів.
- 🌐 Дві мови: українська та англійська.
- 🦸 Персонажі, що еволюціонують із рівнем (включно з «Притулком» ⛺→🏰).

## 🛠 Технічно

- Один файл `index.html` — чистий JavaScript без фреймворків.
- Працює як **Telegram Mini App** (синхронізація через Telegram CloudStorage) і як звичайний сайт (localStorage).
- **PWA**: `manifest.json` + `sw.js` — застосунок можна встановити на телефон («Додати на головний екран»), працює офлайн.
- AI-функції — через Cloudflare Worker, який проксіює Anthropic API (ключ не зберігається в коді).
- `.github/workflows/reminder.yml` — щоденне нагадування в Telegram о ~9:00 за Києвом.

## 🚀 Запуск

Сайт хоститься на GitHub Pages: увімкни **Settings → Pages → Deploy from branch → main**, і застосунок буде доступний за адресою `https://kovalcam-ux.github.io/shlyah-do-mriyi/`.

Локально: просто відкрий `index.html` у браузері (для роботи service worker потрібен http-сервер, напр. `npx serve`).
